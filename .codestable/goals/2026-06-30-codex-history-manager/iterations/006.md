## Problems Encountered

- `build-review-packet.py` 初次执行在 Windows 控制台编码下崩溃：
  - 根因：`subprocess.run(text=True)` 默认使用本机 `gbk` 解码 git diff，遇到 UTF-8 字符后 `stdout` 读取线程报 `UnicodeDecodeError`

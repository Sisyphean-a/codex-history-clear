# Iteration 001

## Current Understanding

owner 要把当前三栏清理界面改成更收敛的双栏工具界面：顶部集中承接操作和清理摘要，左侧只保留精简后的概览与控制，右侧保留会话列表与高级详情，删除动作改成 `DELETE` 二次确认弹窗。

## Implementation Approach

- 删掉独立“本地清理计划”整栏，把计划摘要和主动作并入顶部操作区。
- 去掉策略按钮与说明文案，保留筛选条件和安全选项。
- 新增删除预览弹窗，在真正执行前展示计划表、风险提示和二次确认输入。
- 重写主界面样式，压缩成更像开发工具的深色双栏布局。

## Changes This Iteration

- 改页面结构：主工作区从三栏改成左侧控制栏 + 右侧主内容，顶部新增操作摘要区。
- 改顶部交互：保留更换目录、打开备份目录、重新扫描；新增将删除、预计释放、备份位置和“删除预览”入口；执行后在顶部保留恢复与导出入口。
- 改概览与控制：概览收缩成总会话、已归档、重复项、扫描对象、未识别对象；控制区删掉策略按钮和说明文案，只保留状态、目录、时间、大小与全部安全项。
- 改删除流程：确认词从 `CLEAN` 切到 `DELETE`；删除预览改为弹窗，底部保留取消、只备份不删除、确认删除。
- 改视觉系统：从浅色卡片看板改成低饱和深色开发工具风，并同步重写响应式布局。

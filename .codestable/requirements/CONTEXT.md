# Codex 会话历史治理

本 context 描述“Windows 上治理 Codex 本地会话历史”这件事里会反复出现的专有概念。它只定义术语，不承载实现方案。

## Language

### 存储对象

**CODEX_HOME**:
Codex 本地状态的用户级根目录，默认位于当前用户目录下的 `.codex`。
_Avoid_: 安装目录, 程序目录

**全局历史**:
跨会话的追加式消息历史文件，典型对象是 `history.jsonl`。
_Avoid_: 会话转录, 会话索引

**会话转录**:
按单会话保存消息序列的 JSONL 文件，典型对象是 `rollout-*.jsonl`。
_Avoid_: 全局历史, 诊断日志

**活动数据库**:
当前客户端正在占用的 SQLite 状态数据库对象。
_Avoid_: 备份副本, 导出快照

**候选根目录**:
在真实存储未确认前需要探测的一组本地根路径。
_Avoid_: 真源目录, 权威目录

### 治理动作

**删除计划**:
在 destructive 动作前生成的结构化动作清单。
_Avoid_: 运行脚本, 临时命令

**隔离区**:
待删除对象进入永久清除前的本地暂存位置。
_Avoid_: 回收站, 最终归档

**回滚日志**:
记录 destructive 前后路径映射和执行结果的审计文件。
_Avoid_: 运行日志, 调试输出

**保留本**:
同一重复组里被选为权威保留对象的那一份记录或文件。
_Avoid_: 任意副本, 最新文件

### 路径与重复

**路径别名**:
指向同一工程或同一物理对象的不同路径表示，例如 Win32、WSL、junction 或 symlink 入口。
_Avoid_: 物理重复, 真复制

**逻辑重复**:
指向同一会话内容但可能来自不同索引、不同展示入口或不同路径表示的重复记录。
_Avoid_: 物理重复

**物理重复**:
真实复制出的多份文件对象。
_Avoid_: 路径别名, 索引漂移

### 应用壳与运行

**桌面壳层**:
基于 Wails2 的本地桌面宿主，负责承载前端界面、窗口生命周期和后端绑定。
_Avoid_: 浏览器页面, 云端控制台

**执行任务**:
以 `run_id` 标识的一次扫描、计划或执行作业实例，以及对应的进度与证据集合。
_Avoid_: 单个命令, 临时操作
